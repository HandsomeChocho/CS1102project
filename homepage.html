Image Compression Algorithms

Lossy

- Lossy compression significantly reduces file size by eliminating certain information. The algorithm detects some elements considered less important and can be removed from the image
  • Advantage: Small file sizes. Ideal for web use. Lots of tools, plugins and software support it. Fast to process
  • Disadvantage： Quality degrades due to higher rate of compression.

    • Lossless

    - Lossless Definition: Lossless compression allows perfect reconstruction from compressed data by exploiting statistical redundancy without losing any information.
  • Advantage: Lossless compression preserves all original data which retains the quality of the file.
  • Disadvantage： Lossless compression has a lower compression ratio, meaning the compressed file size will be larger than lossy compression.
